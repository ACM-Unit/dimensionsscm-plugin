If Project scope is specified, then a project baseline is created.

If Part scope is specified, then a design part scoped baseline is created.

If Revised scope is specified, then a revised baseline is created.

(Note: Using a project scoped baseline causes the plug-in to ignore the 'Owning part for baseline' and 'Baseline Template' fields, even if they have been given specific values in the configuration; also a project baseline is always owned by the product that owns the stream or project).
